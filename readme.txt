1) Скачать проект
2) Создать вирутальную среду в папке с проектом 
3) Активировать виртуальную среду
4) Установить библиотеки командой (для Windows):
pip install -r requirements.txt
5) Установить переменную приложения (для Windows):
set FLASK_APP=test-tester.py
6) Запустить проект командой:
flask run
7) Убедиться, что сайт открывается по ссылке http://127.0.0.1:5000/ или http://localhost:5000/  и ошибок нет
8) Создать на свое усмотрение 2 юнит-теста
9) Создать на свое усмотрение 3 авто-теста для интерфейса с помощью selenium или другого фреймворка на python
